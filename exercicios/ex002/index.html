Paragrafos e quebras de linha

Para quebrar a linha use a tag br, assim

Para criar um paragafo basta colocar a tag <p>.

Para escrever os caracteres <> no site sem fazer com que ele altere o código, basta colocar um & junto com o lt para o < e para o > coloque o & junto com o gt.

Vamos adicionar alguns símbolos especiais: ® ¢ ℜ € £ ™

Para adicionar esses símbolos especiais basta abrir uma tag de parágrafo e colocar um & e depois pesquisar pelo nome do símbolo que você quer pesquisar, tem o do euro, do centavo americano (cent), do iene(yen), etc...

Vamos adicionar alguns emojis: 😎 😽

Para adicionar esses emojis, basta entrar na emojipedia no google, encontrar o emoji que você quer, clica nele, rola a tela até encontrar um código como esse U+1F63D, copie só o que vem depois do +, no visual code, abra um uma tag de parágrafo e digite &#x, cole o código e atualize o site.
